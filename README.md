##  Proyectos Destacados

| Proyecto | Tecnologías | Descripción |
|----------|-------------|-------------|
| [**API E-commerce**](Proyectos/API-Ecommerce-Spring) | Spring Boot, JPA, H2 | API REST completa: carrito de compra, pedidos, envíos, wishlist y gestión de productos con DTOs y capa de servicios |
| [**API REST Tareas**](Proyectos/API-REST-Tareas) | Spring Boot, REST | Sistema de gestión de tareas familiares con asignaciones, categorías y estados |
| [**DAO Northwind**](Proyectos/DAO-Northwind) | JDBC, HikariCP | Arquitectura limpia con patrón DAO/Service/DTO, pool de conexiones HikariCP y tests de integración |
| [**Tetris**](Proyectos/Tetris-Java-Swing) | Java Swing, POO | Juego Tetris completo: game loop, 7 tipos de piezas con herencia, rotación y renderizado |
| [**Juego Multijugador**](Proyectos/Juego-Sockets-Multijugador) | Sockets TCP, Multihilo | Juego cliente-servidor con hilo dedicado por jugador y estado compartido sincronizado |
| [**Módulo Odoo Gimnasio**](Proyectos/Modulo-Odoo-Gimnasio) | Python, Odoo ORM | Módulo Odoo completo: modelos, vistas XML, controladores web, seguridad y datos demo |
| [**Módulo Odoo Parking**](Proyectos/Modulo-Odoo-Parking) | Python, Odoo ORM | Sistema de gestión de parking como módulo Odoo con modelos y vistas |

> Cada proyecto contiene su propio **README.md** con detalles técnicos, estructura y cómo ejecutarlo.

##  Ejercicios por Tecnología

###  [Java](Ejercicios-Java/) — 13 categorías

| Categoría | Contenido |
|-----------|-----------|
| POO | Clases, herencia, polimorfismo, clases abstractas, enums |
| Colecciones | ArrayList, LinkedList, conjuntos, Comparable, iteradores |
| Excepciones | try-catch, throw, jerarquía de excepciones |
| Ficheros & IO | Lectura/escritura, serialización, Scanner, DataIO |
| Programación Funcional | Lambdas, interfaces funcionales |
| JDBC | Conexiones, PreparedStatement, patrón DAO |
| Hibernate/JPA | Mapeo de entidades, consultas, informes |
| Spring Boot | REST controllers, CRUD, Swing + BBDD |
| Multihilo | Creación de hilos, sincronización, productor-consumidor |
| Sockets | TCP, UDP, cliente-servidor, multihilo |
| Cifrado | AES, gestión de claves |
| XML | Procesamiento DOM y SAX |

###  [Android](Ejercicios-Android/) — 11 aplicaciones
Ciclo de vida, SQLite, juegos, localización, UI components.

###  [Angular](Ejercicios-Angular/)
Ejercicios de TypeScript básico y aplicación Country App.

###  [Visual Basic .NET](Ejercicios-VisualBasic/) — 4 aplicaciones WinForms
Agenda de contactos (import/export), calculadora científica, sorteo aleatorio, mezclador de colores RGB con timers.

###  [Web](Ejercicios-Web/)
HTML/CSS, JavaScript DOM, algoritmia (ProgramaMe), PHP.
